# Archived: Industry Trackers (SEC + News + Weekly LLM Synthesis)

This folder contains the previous **industry trackers** product: SEC 10-Q/10-K strategy extraction, RSS news ingestion, and LLM-generated weekly summaries by industry. It is **archived** and **not part of the current site launch**.

- **Run from here** (if needed): `cd _archive/industry_trackers && python run_industry_tracker.py consumer --pipelines sec,news,weekly` (with venv and dependencies from repo root).
- **Outputs:** `outputs/industry_trackers/` inside this directory.
- The live site now features the **comparables tracker** (valuation and operational metrics by industry, updated monthly) instead.
